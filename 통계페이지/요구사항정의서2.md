# 📊 친환경차 주차 및 충전시설 관리 대시보드 요구사항 정의서

## 1. 프로젝트 개요
* **목적**: 전용주차구역 및 충전시설 의무설치 이행여부 실시간 모니터링 및 조사 현황 관리
* **핵심 타겟**: 관리자 (모바일 현장 확인 및 데스크톱 보고용)
* **주요 기술 요구사항**: 
    * **Excel 데이터 연동**: 업로드된 엑셀 시트를 기반으로 실시간 수치 업데이트
    * **Mobile First**: 스마트폰 환경에서 모든 그래프와 표가 최적화되어 출력

---

## 2. 디자인 및 UI/UX 가이드 (벤치마킹 반영)

### 🎨 시각적 요소 (`대시보드.jpg` 스타일)
* **메인 컬러**: 깨끗한 **White 배경**에 **Mint/Green 포인트** 컬러 사용 (친환경 이미지 강조)
* **카드 레이아웃**: 각 지표를 독립적인 라운드 코너 카드로 분리하여 가독성 확보
* **아이콘**: 시설구분 및 이행 여부를 직관적으로 알 수 있는 플랫 아이콘 활용
* **타이포그래피**: 굵고 명확한 숫자를 사용하여 핵심 지표(이행률 등)를 최상단에 배치

### ⚙️ 기능적 요소 (`보드.webp` 스타일)
* **상태 요약 바**: 상단에 CPU/Memory 점유율 대신 **'전체 이행률', '미이행 시설 수'** 등을 프로그레스 바 형태로 상시 노출
* **알림 시스템**: 의무 미이행 시설 발생 시 우측 상단 알림 아이콘 및 리스트 피드 생성
* **인터랙티브 차트**: 마우스 오버(또는 터치) 시 상세 수치가 팝업되는 다이내믹 그래프

---

## 3. 상세 페이지 구성 (Wireframe)

### 3.1. 상단 필터 영역 (Search & Filter)
* **시설구분 선택**: [전체 / 공공 / 아파트 / 기상업시설 등] 드롭다운
* **지역 및 상세 위치**: 관리 구역별 필터링
* **조사 기간**: 주 단위(Weekly) 조회를 위한 날짜 범위 선택기
* **데이터 업데이트 버튼**: "엑셀 파일 업로드" 기능을 통해 데이터 즉시 쇄신

### 3.2. 핵심 지표 영역 (KPI Cards)
| 구분 | 전용주차구역 | 충전시설 |
| :--- | :--- | :--- |
| **이행률** | 도넛 차트 (71% 형태) | 도넛 차트 (43% 형태) |
| **현황** | 총 수량 / 이행 / 미이행 (숫자) | 총 수량 / 이행 / 미이행 (숫자) |
| **증감** | 전주 대비 변동폭 (▲3% w/w) | 전주 대비 변동폭 (▼1% w/w) |

### 3.3. 데이터 시각화 영역 (Main Charts)
* **주차/충전시설 이행 추이**: 꺾은선 그래프(Line Chart)를 통해 주 단위 이행률 변화 표시
* **시설구분별 조사 진행률**: 막대 그래프(Bar Chart)로 '공공기관', '공동주택' 등 카테고리별 조사 완료 현황 비교
* **미이행 사유 통계**: 수평 막대 그래프를 통해 미이행 원인(예: 공간 부족, 예산 미확보 등) 순위 표시

### 3.4. 하단 상세 리스트 (Data Table)
* **구성**: 시설명, 시설구분, 의무수량, 설치수량, 이행여부(Status Tag), 최종 조사일
* **기능**: 모바일 환경에서는 '카드형 리스트'로 전환되어 세로 스크롤 최적화
