# 📊 친환경차 시설 관리 대시보드 기술 명세서 (v1.1)

## 1. 데이터 업데이트 및 동기화 설정
* **실시간 일자 반영**: 엑셀 파일의 `최종 수정일(Metadata)` 또는 파일 내 기록된 `최종 조사일` 데이터를 추출합니다.
* **표시 위치**: 대시보드 **오른쪽 상단 (Header 영역)**
* **표기 형식**: `최근 업데이트: 2026-04-25 10:46`
* **연동 방식**: 엑셀 파일 업로드 시 시스템이 자동으로 날짜를 읽어와 대시보드 전체 수치를 갱신합니다.

---

## 2. 엑셀 데이터 파싱 구조 (통합관리 시트 기준)
대시보드 구현을 위해 시트 내의 다음 필드들을 핵심 지표로 연결합니다.

| 엑셀 컬럼명 | 대시보드 활용 항목 | 비고 |
| :--- | :--- | :--- |
| **시설구분** | 상단 필터 및 그룹화 | 공공건물, 기상업시설 등 구분 |
| **주차_의무수량** | 목표치(Goal) 설정 | 이행률 계산의 분모 |
| **주차_설치수량** | 현재값(Current) 설정 | 이행률 계산의 분자 |
| **충전_의무수량** | 목표치(Goal) 설정 | $이행률(\%) = \frac{설치}{의무} \times 100$ |
| **충전_설치수량** | 현재값(Current) 설정 | |
| **이행여부** | 상태 카드 (Status) | 미이행 시설 집중 모니터링 |

---

## 3. 화면 구성 및 디자인 요소 (UI/UX)

### 🟢 상단 헤더 (Top Bar)
* **좌측**: 대시보드 타이틀 (친환경차 시설 관리 시스템)
* **우측**: 📅 **업데이트 일자 표시** (파일 저장 일자 기준)

### 🔍 검색 및 필터 (Filter - 상단 배치)
* `시설구분` 멀티 선택 드롭다운
* `이행여부` 필터 (전체/이행/미이행)
* `시설명` 직접 검색 창

### 📈 메인 대시보드 카드 (Main Visual)
> **디자인 벤치마킹**: `대시보드.jpg`의 밝은 배경과 녹색 포인트 활용

1.  **전용주차구역 현황 카드**
    * 총 의무 수량 대비 설치 수량 실시간 집계
    * **이행률 %**를 중앙에 크게 표시 (애니메이션 효과)
    * 전주 대비 설치 수량 증감 폭 표시
2.  **충전시설 현황 카드**
    * 총 의무 수량 대비 설치 수량 실시간 집계
    * **이행률 %** 표시
3.  **조사 진행률 (Weekly)**
    * 주 단위로 업데이트되는 엑셀 행(Row) 개수를 파악하여 전체 대비 조사 완료 비중 시각화

---

## 4. 모바일 최적화 상세 (Mobile UI)
* **Stacking Layout**: PC에서의 가로 배열 카드들을 모바일에서는 세로로 1열 배치합니다.
* **Floating Action Button (FAB)**: 화면 하단에 엑셀 파일을 즉시 업로드할 수 있는 (+) 버튼을 배치하여 현장에서도 바로 업데이트가 가능하게 구성합니다.
* **Micro-interaction**: 미이행 시설 리스트 클릭 시, 상세 정보가 하단에서 위로 올라오는 '바텀 시트(Bottom Sheet)' 구조를 채택합니다.

---

## 5. 기술적 구현 제안
* **데이터 처리**: `Python(Pandas)` 라이브러리를 사용하여 엑셀의 '통합관리' 시트를 읽고 이행률을 자동 계산합니다.
* **시각화**: `대시보드.jpg`와 유사한 깔끔한 UI 구현을 위해 `Chart.js` 또는 `Tailwind CSS` 기반의 컴포넌트를 사용합니다.
